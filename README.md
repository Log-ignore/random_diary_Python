# random_diary_Python
## 説明
webアプリ「乱数日記」の自作版です。言語にはPythonを使用しています。
前回AIに作らせたhtmlの乱数日記は選択肢をhtmlで直にいじらないといけませんでしたが、
これは自分が選択肢を自由に追加することができます。
（その代わり、選択肢リストのファイルが必要です。）

## 使い方
### 追加したい場合
"when","where","who","how","do"のいずれかを入力したあと、
選択肢として追加したい文節を入力すると、対応した選択肢リストに選択肢が追加されます。

### 文を生成したい場合
"generate"と入力してください。
文が選択肢リストからランダムに生成されます。
